Integrante:
Nome: Gabriel Oliveira de Santana

Descrição do Projeto:
Este projeto é uma aplicação móvel desenvolvida em React Native (Expo) como solução para o caso de uso "SkillUpPlus 2030+". 
O aplicativo funciona como uma plataforma de cursos de tecnologia (IA, Cloud e Computação Quântica), permitindo ao usuário se cadastrar, fazer login, acompanhar o progresso das aulas e editar seu perfil.

Principais recursos implementados:
Navegação Híbrida (Stack Navigation + Bottom Tabs).
Persistência de dados locais (Login, Cadastro e Perfil) utilizando `AsyncStorage`.
Controle de estado global para inscrição e conclusão de cursos.
Validação de formulários de acesso.
